# FITME — 상세 체형 분석 플랫폼 블루프린트

## 1. 프로젝트 개요
FITME은 단순한 키와 몸무게를 넘어, 신체의 세부 비율을 분석하여 사용자에게 최적화된 체형 리포트와 스타일 가이드를 제공하는 프리미엄 플랫폼입니다.

## 2. 주요 기능 및 디자인
- **세부 부위 측정 (Hotspot UI):** 인체 모델의 특정 부위를 클릭하여 상세 치수를 입력.
- **다국어 지원:** 한국어, 영어, 일본어, 중국어 지원.
- **실시간 시각적 피드백 (v1.2):** 입력된 치수에 따라 인체 SVG 모델의 비율이 실시간으로 변화.
- **개인화된 스타일 추천 (v1.2):** 체형 분석 결과에 따른 베스트/워스트 스타일링 카드 제공.
- **데이터 영속성 (v1.2):** LocalStorage를 사용하여 사용자 치수 데이터 자동 저장 및 유지.
- **결과 공유 및 저장 (v1.2):** 분석 리포트 이미지 저장 및 공유 기능.

## 3. 구현된 스타일 및 특징
- **Aesthetics:** 다크 모드 기반의 프리미엄 및 택타일 디자인.
- **Interactive SVG:** CSS Transform과 JS를 이용한 동적 인체 모델링.
- **Mobile Optimized:** 모바일 자동 스크롤 및 터치 최적화 UI.
